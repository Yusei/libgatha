Reads a two-player game from a text file.

File format:

line 1: number of players (p)

line 2: number of strategies (s)

line 3..(3+p): s elements containing p payoffs, separated by ';' the payoffs are separated by ','

Retrieves the payoff of a player, given a set of strategy choices.

Remarks:
For a two-player game with 5 strategies, the payoff for player 1, when 1 plays 3 and 2 plays 4 is given by: gatha_payoff_matrix(m, 1, 3, 4);
For a 3-player game with 5 strategies, the payoff for player 1, when 1 plays 3, 2 plays 4 and 3 plays 1 is given by: gatha_payoff_matrix(m, 1, 3, 4, 1);
Parameters:

...The strategy choices, given as a variable length list of integers.
