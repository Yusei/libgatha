Checks for convergence within an interval.

Assumes we reached convergence if the payoffs stayed T iterations in an interval of size S. N and T are stored in a GathaIntervalData object.

Parameters:
payoffsThe current interation payoff array, containing dp->n_players elements.
dpA pointer to a GathaIntervalData object.
Returns:
TRUE if the algorithm converged, FALSE otherwise.
