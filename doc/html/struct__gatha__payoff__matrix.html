Detailed Description

Normal Form game.

The payoffs for all possible choices of strategies are stored in a matrix of dimension n_players, that contains n_strategies^(n_players+1) elements.

Field Documentation

payoff_t GathaPayoffMatrix::max_payoff

Maximum payoff in the matrix.

It is used, for example, for turning a gain into a cost when needed: the cost is then max_payoff-actual_payoff.

int GathaPayoffMatrix::n_players

Number of players in the game.

This is also the dimension of the matrix.

int GathaPayoffMatrix::n_strategies

Number of strategies for each player.

It is assumed that all players have the same number of strategies.

payoff_t* GathaPayoffMatrix::payoffs

Payoffs for all possible choices of strategies.

The array should not be accessed directly, but through accessors like gatha_payoff_matrix_get and _set.
